# VORTEX-TECH-week-2-remote-internship
Here is the week 2 task for remote internship at VORTEX TECH  in the domain of cybersecurity.


## Objective

This project demonstrates:

- Password Strength Checker in Python
- Basic Network Port Scanning using Nmap

## Files

password_checker.py
Python script that evaluates password strength.

port_scan_report.md
Report explaining the Nmap scan results.

## Requirements

Python 3.x

Nmap

## How to Run

Run the password checker:

python password_checker.py

Run the port scan:

nmap localhost

or

nmap 127.0.0.1

## Features

✔ Checks password length

✔ Uppercase letters

✔ Lowercase letters

✔ Numbers

✔ Special characters

✔ Common password detection

✔ Password strength rating
